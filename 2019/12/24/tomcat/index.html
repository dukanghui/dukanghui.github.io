JDK安装

  1. 查看目前系统的jdk: rpm -qa | grep jdk
  2. 查看yum库中有哪些JDK: yum search java|grep jdk
  3. 选择要安装的版本的包名,进行安装: yum install java-1.7.0-openjdk-1.7.0.75.x86_64 (如果已经安装请忽略此步骤)
  4. 也可选择自己要安装的其它版本或最新版本的JDK.注意修改下面配置中的路径。
